  • Не открывайте и, особенно, не сохраняйте файлы в многопользовательском доступе

Итак, давайте посмотрим, что мы можем сделать с приложением :-).

Приложение может быть установлено как просотрщие dbf файлов по умолчанию.

Несколько технических деталей.

  • Это приложение считывает dbf файл базу данных SQLite в память.
    Так, что загрузка больших файлов может быть медленной.
    Все изменеия применяются в базе SQLite, и потом записываются в dbf файл при выборе операции сохранения.
  • Приложение поерживает форматы XBase, dBaseIII, IV, Foxpro 2.x, Visual Foxpro.
  • Следующие типы полей могут редактироваться: character, numeric, currency, double, integer, float, logical.
